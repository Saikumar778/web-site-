Privacy Policy

Last updated: February 2026

This Privacy Policy describes how our mobile application ("MYPA AI") collects, uses, and protects user data. Your privacy is very important to us.

Information We Collect

Our app collects the following information only for user functionality:

  • Daily tasks entered by the user
  • Daily expense and income details entered by the user
  • Notes added by the user

How We Use Your Information

  • To remind users about their daily planned activities
  • To help users track completed tasks
  • To manage daily expenses and earnings
  • To provide offline voice assistant functionality

Data Storage

All user data is stored only on the user's mobile device. This app does not use any cloud storage or external servers.

Internet & Cloud Usage

Our app works completely offline. No user data is uploaded, shared, or synced with any cloud services.

Voice Assistant

The voice assistant feature works fully offline. Voice data is not recorded, stored, or transmitted outside the device.

Data Sharing

We do not share, sell, or transfer user data to any third party.

Data Security

Since all data is stored locally on the user's device, data security depends on the device's security features such as screen lock and encryption.

User Control

Users can edit or delete their data anytime from within the app. Uninstalling the app will permanently delete all stored data.

Children’s Privacy

This app does not knowingly collect personal information from children under 13.
